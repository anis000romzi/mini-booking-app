
## Installation

Clone this repository
```bash
  git clone https://github.com/anis000romzi/mini-booking-app.git
  cd mini-booking-app
```

Install dependency
```bash
  npm install
```

Create environment
```bash
  cp .env.example .env
```

Run the application
```bash
$ npm run dev
```

Build the application
```bash
$ npm run build
```
